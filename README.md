# Miere Cafe Watchdog 🛡️🐕💩
**Version 0.6.9-beta**

## Overview
Miere Cafe Watchdog is a specialized background utility designed to protect essential applications from unauthorized termination. Built with a focus on redundancy and stealth, it ensures that critical shop processes (like timers or security tools if for some reason the thing doesn't feature it) stay alive even if a user tries to "End Task."

## Key Features
* **Persistent Monitoring**: Real-time process health checks with customizable restart delays. 
* **Administrative Security**: Password-protected Tray UI using SHA-256 hashing to prevent unauthorized shutdowns. 
* **Triple-Layer Redundancy**: Config data is synced across hidden system vaults to prevent manual deletion. 
* **Self-Contained**: Bundles its own Java Runtime Environment (JRE) so it works on any PC out of the box.

##🚀 What's New in v0.6.9
Dual-Layer Protection: Combines immediate "Shield" respawns with interval-based "Pulse" launches.

Live Visual Dashboard: A dedicated UI that displays active tasks, their actual .exe icons, and real-time countdowns.

Seconds-First UI: Standardized intervals to seconds for human-readable configuration (30s minimum floor enforced).
